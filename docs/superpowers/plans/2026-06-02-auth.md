# Auth (local + OIDC) — Implementation Plan (Plan 3)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Land the entire authentication surface — Argon2id local credentials, server-side sessions over `sid` cookie, double-submit CSRF, first-run bootstrap, headless admin CLI, OIDC (Dex dev IdP) with PKCE auth-code flow + auto-provisioning policies — behind the API contract in spec §6.1 + §7.

**Architecture:** New `crates/knot-auth` owns the cryptographic primitives (password hash, token gen, login-throttle, OIDC client builder) so they're testable without a server. `crates/knot-storage` grows three concrete stores (`UserStore`, `WorkspaceStore`, `SessionStore`) — async-trait + Postgres impls. `crates/knot-server` adds `auth` middleware module (`SessionLoader`, `RequireSession`, `Csrf`) and `routes::auth` for the seven endpoints in §6.1. CSRF protection runs only on `/api/*` unsafe methods (not on `/auth/*` — login itself bootstraps the session). Authorization (per-document ACL resolver) is deferred to Plan 4 — Plan 3 only resolves *workspace* role.

**Tech Stack:** `argon2` 0.5, `rand` 0.8 (`OsRng`), `base64` 0.22, `openidconnect` 4.x with `reqwest` 0.12 client, `cookie` 0.18, `axum` middleware (tower::Layer), `hmac` 0.12 + `sha2` 0.10 for CSRF token signing, `clap` 4 with derive for the admin subcommand router.

**Predecessor:** Plan 2 (Repo Bootstrap & DB, tag `plan-2-complete` at `2a4a0f5`). `workspaces`, `users`, `workspace_members`, `sessions` tables exist; nothing reads or writes them.

**Out of scope for this plan** (each gets its own later plan):
- Per-document ACL resolver + `document_grants` queries (Plan 4)
- ACL cache + `acl_invalidations` NOTIFY listener (Plan 4)
- Workspace members CRUD endpoints (`PATCH /api/workspace/members/:id`, etc.) (Plan 4)
- Frontend login UI shell (Plans 6-8) — e2e drives the API directly
- Account-locking / MFA / passkey (post-v0.1)

## Conventions for this plan

- **Argon2id params:** prod = `Argon2::default()` (m=19456 KiB, t=2, p=1 — OWASP 2023). Tests override via `Auth::new_with_params(...)`. The spec's m=64MiB is a target for the cost-benchmark phase but slows CI unacceptably; defaults are within OWASP guidance and revisitable.
- **Time:** `chrono::Utc::now()` everywhere — no `std::time::SystemTime`. Token expiry is computed as `now + Duration::days(30)`.
- **Random:** `rand::rngs::OsRng` for tokens and salt. Never `thread_rng`.
- **Errors:** every store/auth crate uses `thiserror`. Server routes map errors to the spec §6.3 envelope:
  ```rust
  // crates/knot-server/src/routes/error.rs already-pattern from health.rs:
  // Just inline the envelope inside each handler; no separate error type in Plan 3.
  // Codes: "auth.invalid_credentials", "auth.session_expired", "auth.setup_closed",
  //        "auth.csrf", "auth.oidc.state_mismatch", "auth.oidc.exchange_failed".
  ```
- **Cookies:** Always `HttpOnly`, `Path=/`. `Secure` + `SameSite=Lax` in prod (`base_url` starts with `https://`); drop `Secure` in dev so localhost works without TLS.
- **Tests:**
  - `knot-auth`: pure unit tests; no DB, no I/O.
  - `knot-storage`: testcontainers integration tests (one per store) under `tests/`.
  - `knot-server`: in-process Tower app tests via `axum::Router::oneshot` + a `Mock*Store` trait impl. One DB-backed integration test per auth route under `tests/auth_integration.rs` using `testcontainers`.
  - `e2e`: one full happy-path flow.

```rust
//! Argon2id password hashing.
//!
//! Uses the `argon2` crate's default parameters (m=19456 KiB, t=2, p=1)
//! which match the 2023 OWASP guidance. Tests override via
//! `Hasher::with_params` to keep CI fast.

use argon2::{
    Argon2,
    password_hash::{PasswordHash, PasswordHasher, PasswordVerifier, SaltString, rand_core::OsRng},
};
use thiserror::Error;

#[derive(Debug, Error)]
pub enum PasswordError {
    #[error("hash: {0}")]
    Hash(String),
    #[error("invalid password format")]
    Format,
}

#[derive(Clone)]
pub struct Hasher {
    argon: Argon2<'static>,
}

impl Hasher {
    pub fn new() -> Self {
        Self {
            argon: Argon2::default(),
        }
    }

    #[cfg(any(test, feature = "fast-params"))]
    pub fn fast_for_tests() -> Self {
        // Minimum legal Argon2id params: m=8 KiB, t=1, p=1. ~5 ms on CI.
        use argon2::{Algorithm, Params, Version};
        let params = Params::new(8, 1, 1, None).expect("params");
        Self {
            argon: Argon2::new(Algorithm::Argon2id, Version::V0x13, params),
        }
    }

    pub fn hash(&self, plain: &str) -> Result<String, PasswordError> {
        let salt = SaltString::generate(&mut OsRng);
        self.argon
            .hash_password(plain.as_bytes(), &salt)
            .map(|h| h.to_string())
            .map_err(|e| PasswordError::Hash(e.to_string()))
    }

    pub fn verify(&self, hashed: &str, plain: &str) -> Result<bool, PasswordError> {
        let parsed = PasswordHash::new(hashed).map_err(|_| PasswordError::Format)?;
        Ok(self.argon.verify_password(plain.as_bytes(), &parsed).is_ok())
    }
}

impl Default for Hasher {
    fn default() -> Self {
        Self::new()
    }
}

#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn hash_then_verify_roundtrip() {
        let h = Hasher::fast_for_tests();
        let hashed = h.hash("hunter2").expect("hash");
        assert!(h.verify(&hashed, "hunter2").expect("verify"));
```

```rust
//! 32-byte cryptographic session token + base64url codec.
//!
//! Tokens are produced by the OS CSPRNG. Storage uses the raw 32 bytes
//! (as `bytea` in the `sessions` table); the cookie carries the
//! base64url-encoded form (43 chars, no padding).

use base64::{Engine, engine::general_purpose::URL_SAFE_NO_PAD};
use rand::RngCore;
use thiserror::Error;

pub const TOKEN_BYTES: usize = 32;

#[derive(Debug, Error)]
pub enum TokenError {
    #[error("malformed token")]
    Malformed,
}

#[derive(Clone, Eq, PartialEq)]
pub struct SessionToken(pub [u8; TOKEN_BYTES]);

impl SessionToken {
    pub fn generate() -> Self {
        let mut buf = [0u8; TOKEN_BYTES];
        rand::rngs::OsRng.fill_bytes(&mut buf);
        Self(buf)
    }

    pub fn encode(&self) -> String {
        URL_SAFE_NO_PAD.encode(self.0)
    }

    pub fn decode(s: &str) -> Result<Self, TokenError> {
        let bytes = URL_SAFE_NO_PAD.decode(s).map_err(|_| TokenError::Malformed)?;
        if bytes.len() != TOKEN_BYTES {
            return Err(TokenError::Malformed);
        }
        let mut buf = [0u8; TOKEN_BYTES];
        buf.copy_from_slice(&bytes);
        Ok(Self(buf))
    }

    pub fn as_bytes(&self) -> &[u8] {
        &self.0
    }
}

    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {

    }
}
```

```rust
//! Per-key login throttling — leaky bucket with capacity 5, drains 1/min.
//!
//! Tracks two independent keyspaces from the caller: IP and email. After 5
//! failures within 5 min the throttle returns `Allow::Deny`. Successful
//! logins do NOT touch the throttle (Plan 3 keeps it dumb on purpose).
//!
//! Time is injected via the `Clock` trait so tests don't need `tokio::time`.

use std::collections::HashMap;
use std::sync::Mutex;
use std::time::{Duration, Instant};

#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum Allow {
    /// Caller may proceed with the authentication attempt.
    Yes,
    /// Caller is over budget; return the generic "invalid credentials"
    /// response. The handler should also sleep 1 s before responding.
    No,
}

pub trait Clock: Send + Sync + 'static {
    fn now(&self) -> Instant;
}

pub struct SystemClock;
impl Clock for SystemClock {
    fn now(&self) -> Instant {
        Instant::now()
    }
}

const CAPACITY: u32 = 5;
const DRAIN_PER: Duration = Duration::from_secs(60); // 1 token / minute

struct Bucket {
    tokens: u32,
    last_drained: Instant,
}

pub struct Throttle<C: Clock = SystemClock> {
    clock: C,
    buckets: Mutex<HashMap<String, Bucket>>,
}

impl Throttle<SystemClock> {
    pub fn new() -> Self {
        Self {
            clock: SystemClock,
            buckets: Mutex::new(HashMap::new()),
        }
    }
}

impl Default for Throttle<SystemClock> {
    fn default() -> Self {
        Self::new()
    }
}

impl<C: Clock> Throttle<C> {
    pub fn with_clock(clock: C) -> Self {
        Self {
            clock,
            buckets: Mutex::new(HashMap::new()),
        }
    }

    /// Check whether `key` may attempt a login. Does NOT record a failure;
    /// the caller invokes `record_failure` after a failed credential check.
    pub fn check(&self, key: &str) -> Allow {
        let mut map = self.buckets.lock().expect("throttle mutex");
        let now = self.clock.now();
        let drained = drain(map.get(key), now);
        if drained >= CAPACITY {
            Allow::No
        } else {
            Allow::Yes
        }
    }

    /// Record a failed login for `key`. Returns the new failure count
    /// (1..=CAPACITY, capped).
    pub fn record_failure(&self, key: &str) -> u32 {
        let mut map = self.buckets.lock().expect("throttle mutex");
        let now = self.clock.now();
        let drained = drain(map.get(key), now);
        let new_count = (drained + 1).min(CAPACITY);
        map.insert(
            key.to_string(),
            Bucket {
                tokens: new_count,
                last_drained: now,
            },
        );
        new_count
    }

    /// Reset a key (called on successful login).
    pub fn reset(&self, key: &str) {
        let mut map = self.buckets.lock().expect("throttle mutex");
        map.remove(key);
    }
}

fn drain(b: Option<&Bucket>, now: Instant) -> u32 {
    let Some(b) = b else { return 0 };
    let elapsed = now.saturating_duration_since(b.last_drained);
    let drained = (elapsed.as_secs() / DRAIN_PER.as_secs()) as u32;
    b.tokens.saturating_sub(drained)
}
```

```rust
#[derive(Debug, Clone, PartialEq, Eq)]
pub struct Session {
    pub id: Vec<u8>,
    pub user_id: Uuid,
    pub workspace_id: Uuid,
    pub created_at: DateTime<Utc>,
    pub expires_at: DateTime<Utc>,
    pub last_seen_at: DateTime<Utc>,

    #[error("sqlx: {0}")]
    Sqlx(#[from] sqlx::Error),

pub trait SessionStore: Send + Sync + 'static {
    /// Create a new session. `id` is the raw 32-byte token bytes.
    async fn create(
        &self,
        id: &[u8],
        user_id: Uuid,
        workspace_id: Uuid,

        .fetch_one(&self.pool)
        .await?;

            id: row.0,
            user_id: row.1,
            workspace_id: row.2,
            created_at: row.3,
            expires_at: row.4,
            last_seen_at: row.5,
        })
    }

        let row = sqlx::query_as::<
            _,
            (Vec<u8>, Uuid, Uuid, DateTime<Utc>, DateTime<Utc>, DateTime<Utc>),
        >(
            "SELECT id, user_id, workspace_id, created_at, expires_at, last_seen_at
             FROM sessions WHERE id = $1 AND expires_at > now()",
        )
        .bind(id)
        .fetch_optional(&self.pool)
        .await?;
        Ok(row.map(|r| Session {
            id: r.0,
            user_id: r.1,
            workspace_id: r.2,
            created_at: r.3,
            expires_at: r.4,
            last_seen_at: r.5,
        }))
    }

    async fn touch(&self, id: &[u8]) -> Result<(), SessionStoreError> {
        sqlx::query("UPDATE sessions SET last_seen_at = now() WHERE id = $1")
            .bind(id)
            .execute(&self.pool)
            .await?;
        Ok(())
    }

    async fn delete(&self, id: &[u8]) -> Result<(), SessionStoreError> {
        sqlx::query("DELETE FROM sessions WHERE id = $1")
            .bind(id)
            .execute(&self.pool)
            .await?;

    }
```

```rust
//! Loads the session named by the `sid` cookie and attaches an
//! `AuthContext` to request extensions. No `sid` cookie → no extension,
//! no error: downstream handlers decide whether the route requires auth.

use std::sync::Arc;

use axum::{
    body::Body,
    extract::Request,
    http::header,
    middleware::Next,
    response::Response,
};
use knot_auth::SessionToken;
use knot_storage::{SessionStore, WorkspaceStore};

use super::context::AuthContext;

pub const SID_COOKIE: &str = "sid";

#[derive(Clone)]
pub struct SessionDeps {
    pub sessions: Arc<dyn SessionStore>,
    pub workspaces: Arc<dyn WorkspaceStore>,
}

pub async fn session_loader_mw(
    axum::extract::State(deps): axum::extract::State<SessionDeps>,
    mut req: Request<Body>,
    next: Next,
) -> Response {
    if let Some(token) = extract_sid(&req) {
        if let Ok(decoded) = SessionToken::decode(&token) {
            if let Ok(Some(s)) = deps.sessions.find_active(decoded.as_bytes()).await {
                if let Ok(Some(role)) =
                    deps.workspaces.get_member_role(s.workspace_id, s.user_id).await
                {
                    req.extensions_mut().insert(AuthContext {
                        user_id: s.user_id,
                        workspace_id: s.workspace_id,
                        role,
                    });
                    // Fire-and-forget touch.
                    let sessions = deps.sessions.clone();
                    let id = s.id.clone();
                    tokio::spawn(async move {
                        let _ = sessions.touch(&id).await;
                    });
                }
            }
        }
    }
    next.run(req).await
}

fn extract_sid(req: &Request<Body>) -> Option<String> {
    let header = req.headers().get(header::COOKIE)?.to_str().ok()?;
    for raw in header.split(';') {
        if let Ok(c) = cookie::Cookie::parse(raw.trim()) {
            if c.name() == SID_COOKIE {
                return Some(c.value().to_string());
            }
        }
    }
    None
}
```

```rust
//! Double-submit CSRF tokens. The server mints a token bound to the
//! session id under an HMAC-SHA256 key (the `session_key` config). The
//! client echoes it in an `X-CSRF-Token` header on unsafe-method
//! requests; the server verifies the HMAC matches the cookie.

use base64::{Engine, engine::general_purpose::URL_SAFE_NO_PAD};
use hmac::{Hmac, Mac};
use sha2::Sha256;

pub fn mint(key: &[u8], session_id: &[u8]) -> String {
    let mut mac = Hmac::<Sha256>::new_from_slice(key).expect("hmac key");
    mac.update(session_id);
    let tag = mac.finalize().into_bytes();
    URL_SAFE_NO_PAD.encode(tag)
}

pub fn verify(key: &[u8], session_id: &[u8], token: &str) -> bool {
    let Ok(provided) = URL_SAFE_NO_PAD.decode(token) else {
        return false;
    };
    let mut mac = Hmac::<Sha256>::new_from_slice(key).expect("hmac key");
    mac.update(session_id);
    mac.verify_slice(&provided).is_ok()
}
```

```rust
//! Verify the `X-CSRF-Token` header against the `csrf` cookie for unsafe
//! methods on routes that require an active session. Safe methods
//! (GET/HEAD/OPTIONS) and routes mounted *outside* this layer (e.g. all
//! of `/auth/*`) are unaffected.

use axum::{
    Json,
    body::Body,
    extract::Request,
    http::{Method, StatusCode, header},
    middleware::Next,
    response::{IntoResponse, Response},
};
use serde_json::json;

use super::context::AuthContext;
use super::session_loader::SID_COOKIE;

pub const CSRF_COOKIE: &str = "csrf";
pub const CSRF_HEADER: &str = "x-csrf-token";

pub async fn csrf_mw(req: Request<Body>, next: Next) -> Response {
    let safe = matches!(*req.method(), Method::GET | Method::HEAD | Method::OPTIONS);
    if safe {
        return next.run(req).await;
    }

    // Anonymous unsafe writes can't be CSRF'd by definition — no session to ride.
    if req.extensions().get::<AuthContext>().is_none() {
        return next.run(req).await;
    }

    let csrf_cookie = extract_cookie(&req, CSRF_COOKIE);
    let csrf_header = req
        .headers()
        .get(CSRF_HEADER)
        .and_then(|v| v.to_str().ok())
        .map(|s| s.to_string());

    if let (Some(c), Some(h)) = (csrf_cookie, csrf_header) {
        if c == h {
            return next.run(req).await;
        }
    }

    (
        StatusCode::FORBIDDEN,
        Json(json!({
            "error": {
                "code": "auth.csrf",
                "message": "missing or mismatched CSRF token",
                "details": {}
            }
        })),
    )
        .into_response()
}

fn extract_cookie(req: &Request<Body>, name: &str) -> Option<String> {
    let h = req.headers().get(header::COOKIE)?.to_str().ok()?;
    for raw in h.split(';') {
        if let Ok(c) = cookie::Cookie::parse(raw.trim()) {
            if c.name() == name {
                return Some(c.value().to_string());
            }
        }
    }
    None
}

// Keep SID parsing here too for callers that don't pull session_loader.
pub use SID_COOKIE as _SID;
```

```rust
//! POST /auth/setup — first-run bootstrap. Returns 410 once any user
//! exists. Creates the singleton workspace + first user (owner role).
//!
//! The response sets the `sid` cookie so the operator is logged in.

use axum::{
    Json, Router,
    extract::State,
    http::{HeaderValue, StatusCode, header},
    response::IntoResponse,
    routing::post,
};
use chrono::{Duration, Utc};
use knot_auth::SessionToken;
use serde::{Deserialize, Serialize};
use serde_json::json;

use crate::AppState;
use crate::auth::SID_COOKIE;

#[derive(Deserialize)]
pub struct SetupRequest {
    pub email: String,
    pub password: String,
    pub display_name: String,
    /// Optional workspace name; defaults to "Workspace".
    pub workspace_name: Option<String>,
}

#[derive(Serialize)]
pub struct SetupResponse {
    pub user_id: String,
    pub workspace_id: String,
}

pub fn router() -> Router<AppState> {
    Router::new().route("/auth/setup", post(setup))
}

async fn setup(
    State(state): State<AppState>,
    Json(req): Json<SetupRequest>,
) -> axum::response::Response {
    let users = match &state.users {
        Some(u) => u.clone(),
        None => return error(StatusCode::SERVICE_UNAVAILABLE, "storage", "storage unavailable"),
    };
    let workspaces = match &state.workspaces {
        Some(w) => w.clone(),
        None => return error(StatusCode::SERVICE_UNAVAILABLE, "storage", "storage unavailable"),
    };
    let sessions = match &state.sessions {
        Some(s) => s.clone(),
        None => return error(StatusCode::SERVICE_UNAVAILABLE, "storage", "storage unavailable"),
    };

    if req.password.len() < 8 {
        return error(
            StatusCode::UNPROCESSABLE_ENTITY,
            "auth.weak_password",
            "password must be at least 8 characters",
        );
    }

    let count = match users.count().await {
        Ok(n) => n,
        Err(e) => {
            tracing::error!(error=?e, "setup count");
            return error(StatusCode::INTERNAL_SERVER_ERROR, "internal", "");
        }
    };
    if count > 0 {
        return error(StatusCode::GONE, "auth.setup_closed", "setup already complete");
    }

    // Create workspace.
    let ws_name = req.workspace_name.as_deref().unwrap_or("Workspace");
    let ws = match workspaces.create("default", ws_name).await {
        Ok(w) => w,
        Err(e) => {
            tracing::error!(error=?e, "create workspace");
            return error(StatusCode::INTERNAL_SERVER_ERROR, "internal", "");
        }
    };

    // Hash + create user.
    let hash = match state.hasher.hash(&req.password) {
        Ok(h) => h,
        Err(e) => {
            tracing::error!(error=?e, "hash");
            return error(StatusCode::INTERNAL_SERVER_ERROR, "internal", "");
        }
    };
    let user = match users.create_local(&req.email, &req.display_name, &hash).await {
        Ok(u) => u,
        Err(e) => {
            tracing::error!(error=?e, "create user");
            return error(StatusCode::INTERNAL_SERVER_ERROR, "internal", "");
        }
    };
    if let Err(e) = workspaces
        .add_member(ws.id, user.id, knot_storage::WorkspaceRole::Owner)
        .await
    {
        tracing::error!(error=?e, "add member");
        return error(StatusCode::INTERNAL_SERVER_ERROR, "internal", "");
    }

    // Mint session.
    let token = SessionToken::generate();
    let exp = Utc::now() + Duration::days(30);
    if let Err(e) = sessions
        .create(token.as_bytes(), user.id, ws.id, exp, None, None)
        .await
    {
        tracing::error!(error=?e, "create session");
        return error(StatusCode::INTERNAL_SERVER_ERROR, "internal", "");
    }

    let secure = state.base_url.starts_with("https://");
    let cookie_val = format!(
        "{SID_COOKIE}={}; HttpOnly; SameSite=Lax; Path=/{}",
        token.encode(),
        if secure { "; Secure" } else { "" }
    );

    let mut resp = (
        StatusCode::CREATED,
        Json(SetupResponse {
            user_id: user.id.to_string(),
            workspace_id: ws.id.to_string(),
        }),
    )
        .into_response();
    resp.headers_mut().insert(
        header::SET_COOKIE,
        HeaderValue::from_str(&cookie_val).expect("cookie header"),
    );
    resp
}

fn error(status: StatusCode, code: &str, message: &str) -> axum::response::Response {
    (
        status,
        Json(json!({
            "error": { "code": code, "message": message, "details": {} }
        })),
    )
        .into_response()
}
```

```rust
//!
//! `POST /auth/login`        body: {email,password}    → sets sid cookie
//! `POST /auth/logout`       deletes session row + clears cookie
//! `GET  /auth/session`      → 200 with user+workspace+role, or 401

use std::net::IpAddr;

use axum::{
    Json, Router,
    extract::{ConnectInfo, State},
    http::{HeaderMap, HeaderValue, StatusCode, header},
    response::IntoResponse,
    routing::{get, post},
};
use chrono::{Duration, Utc};
use knot_auth::{Allow, SessionToken};
use serde::{Deserialize, Serialize};
use serde_json::json;
use std::net::SocketAddr;

use crate::AppState;
use crate::auth::{AuthContext, SID_COOKIE};

#[derive(Deserialize)]
pub struct LoginRequest {
    pub email: String,
    pub password: String,
}

#[derive(Serialize)]
pub struct SessionResponse {
    pub user_id: String,
    pub email: String,
    pub display_name: String,
    pub workspace_id: String,
    pub role: String,
}

pub fn router() -> Router<AppState> {
    Router::new()
        .route("/auth/login", post(login))
        .route("/auth/logout", post(logout))
        .route("/auth/session", get(session))
}

async fn login(
    State(state): State<AppState>,
    ConnectInfo(addr): ConnectInfo<SocketAddr>,
    headers: HeaderMap,
    Json(req): Json<LoginRequest>,
) -> axum::response::Response {
    let users = some_or_500!(state.users.clone(), "users");
    let workspaces = some_or_500!(state.workspaces.clone(), "workspaces");
    let sessions = some_or_500!(state.sessions.clone(), "sessions");

    let ip_key = format!("ip:{}", addr.ip());
    let email_key = format!("email:{}", req.email.to_lowercase());
    if matches!(state.throttle.check(&ip_key), Allow::No)
        || matches!(state.throttle.check(&email_key), Allow::No)
    {
        tokio::time::sleep(std::time::Duration::from_secs(1)).await;
        return invalid_credentials();
    }

    let user = match users.find_by_email(&req.email).await {
        Ok(Some(u)) => u,
        Ok(None) => {
            state.throttle.record_failure(&ip_key);
            state.throttle.record_failure(&email_key);
            tokio::time::sleep(std::time::Duration::from_secs(1)).await;
            return invalid_credentials();
        }
        Err(e) => {
            tracing::error!(error=?e, "login lookup");
            return internal();
        }
    };
    let Some(hash) = user.password_hash.as_deref() else {
        state.throttle.record_failure(&email_key);
        tokio::time::sleep(std::time::Duration::from_secs(1)).await;
        return invalid_credentials();
    };

    let ok = match state.hasher.verify(hash, &req.password) {
        Ok(b) => b,
        Err(_) => false,
    };
    if !ok {
        state.throttle.record_failure(&ip_key);
        state.throttle.record_failure(&email_key);
        tokio::time::sleep(std::time::Duration::from_secs(1)).await;
        return invalid_credentials();
    }
    state.throttle.reset(&ip_key);
    state.throttle.reset(&email_key);

    // Look up workspace (singleton v0.1).
    let ws = match workspaces.get_singleton().await {
        Ok(Some(w)) => w,
        _ => return internal(),
    };
    // Membership check.
    match workspaces.get_member_role(ws.id, user.id).await {
        Ok(Some(_)) => {}
        _ => return invalid_credentials(),
    };

    let token = SessionToken::generate();
    let exp = Utc::now() + Duration::days(30);
    let user_agent = headers
        .get(header::USER_AGENT)
        .and_then(|v| v.to_str().ok());
    let ip: IpAddr = addr.ip();
    if let Err(e) = sessions
        .create(token.as_bytes(), user.id, ws.id, exp, user_agent, Some(ip))
        .await
    {
        tracing::error!(error=?e, "create session");
        return internal();
    }
```

```rust
//! Thin wrapper around `openidconnect-rs` for the PKCE authorization-code
//! flow against a single configured IdP. Handles discovery, authorize-URL
//! generation, code exchange, and id_token verification.

use openidconnect::{
    AuthenticationFlow, AuthorizationCode, ClientId, ClientSecret, CsrfToken, IssuerUrl, Nonce,
    PkceCodeChallenge, PkceCodeVerifier, RedirectUrl, Scope,
    core::{CoreAuthenticationFlow, CoreClient, CoreProviderMetadata, CoreResponseType},
    reqwest,
};
use thiserror::Error;
use url::Url;

#[derive(Debug, Error)]
pub enum OidcError {
    #[error("discovery: {0}")]
    Discovery(String),
    #[error("exchange: {0}")]
    Exchange(String),
    #[error("verification: {0}")]
    Verification(String),
    #[error("config: {0}")]
    Config(String),
}

/// Materialised OIDC client. Built once at startup.
pub struct OidcClient {
    pub client: CoreClient,
    pub issuer: String,
}

impl OidcClient {
    pub async fn discover(
        issuer: &str,
        client_id: &str,
        client_secret: &str,
        redirect_url: &str,
    ) -> Result<Self, OidcError> {
        let http = reqwest::Client::builder()
            .redirect(reqwest::redirect::Policy::none())
            .build()
            .map_err(|e| OidcError::Discovery(e.to_string()))?;
        let issuer_url =
            IssuerUrl::new(issuer.to_string()).map_err(|e| OidcError::Config(e.to_string()))?;
        let metadata = CoreProviderMetadata::discover_async(issuer_url, &http)
            .await
            .map_err(|e| OidcError::Discovery(e.to_string()))?;
        let client = CoreClient::from_provider_metadata(
            metadata,
            ClientId::new(client_id.to_string()),
            Some(ClientSecret::new(client_secret.to_string())),
        )
        .set_redirect_uri(
            RedirectUrl::new(redirect_url.to_string())
                .map_err(|e| OidcError::Config(e.to_string()))?,
        );
        Ok(Self {
            client,
            issuer: issuer.to_string(),
        })
    }
}

pub struct FlowStart {
    pub authorize_url: Url,
    pub csrf_state: String,
    pub nonce: String,
    pub pkce_verifier: String,
}

impl OidcClient {
    pub fn build_authorize_url(&self) -> FlowStart {
        let (pkce_challenge, pkce_verifier) = PkceCodeChallenge::new_random_sha256();
        let (auth_url, csrf, nonce) = self
            .client
            .authorize_url(
                CoreAuthenticationFlow::AuthorizationCode,
                CsrfToken::new_random,
                Nonce::new_random,
            )
            .add_scope(Scope::new("openid".into()))
            .add_scope(Scope::new("email".into()))
            .add_scope(Scope::new("profile".into()))
            .add_scope(Scope::new("groups".into()))
            .set_pkce_challenge(pkce_challenge)
            .url();
        FlowStart {
            authorize_url: auth_url,
            csrf_state: csrf.secret().to_string(),
            nonce: nonce.secret().to_string(),
            pkce_verifier: pkce_verifier.secret().to_string(),
        }
    }
}

/// Claims we care about from the id_token.
#[derive(Debug, Clone)]
pub struct VerifiedIdentity {
    pub subject: String,
    pub email: String,
    pub display_name: String,
    pub groups: Vec<String>,
}

impl OidcClient {
    /// Exchange `code` for an id_token. PKCE verifier and `expected_nonce`
    /// must be retrieved from the flow-state cookie.
    pub async fn exchange_code(
        &self,
        code: &str,
        pkce_verifier: &str,
        expected_nonce: &str,
    ) -> Result<VerifiedIdentity, OidcError> {
        let http = reqwest::Client::builder()
            .redirect(reqwest::redirect::Policy::none())
            .build()
            .map_err(|e| OidcError::Exchange(e.to_string()))?;
        let response = self
            .client
            .exchange_code(AuthorizationCode::new(code.to_string()))
            .map_err(|e| OidcError::Exchange(e.to_string()))?
            .set_pkce_verifier(PkceCodeVerifier::new(pkce_verifier.to_string()))
            .request_async(&http)
            .await
            .map_err(|e| OidcError::Exchange(e.to_string()))?;

        let id_token = response
            .id_token()
            .ok_or_else(|| OidcError::Verification("missing id_token".into()))?;
        let nonce = Nonce::new(expected_nonce.to_string());
        let id_verifier = self.client.id_token_verifier();
        let claims = id_token
            .claims(&id_verifier, &nonce)
            .map_err(|e| OidcError::Verification(e.to_string()))?;

        let email = claims
            .email()
            .ok_or_else(|| OidcError::Verification("missing email claim".into()))?
            .to_string();
        let display_name = claims
            .name()
            .and_then(|n| n.get(None))
            .map(|n| n.to_string())
            .unwrap_or_else(|| email.clone());
        let groups: Vec<String> = claims
            .additional_claims()
            .groups
            .clone()
            .unwrap_or_default();

        Ok(VerifiedIdentity {
            subject: claims.subject().to_string(),
            email,
            display_name,
            groups,
        })
    }
}
```

```rust
    );

    let mut resp = Redirect::to(start.authorize_url.as_str()).into_response();
    resp.headers_mut()
        .insert(header::SET_COOKIE, HeaderValue::from_str(&cookie_val).unwrap());
    resp
}

async fn callback(
    State(state): State<AppState>,
    headers: HeaderMap,
    Query(q): Query<CallbackQuery>,
) -> axum::response::Response {
    let Some(oidc) = state.oidc.clone() else {
        return (StatusCode::SERVICE_UNAVAILABLE, "oidc disabled").into_response();
    };
    let users = match state.users.clone() {
        Some(u) => u,
        None => return (StatusCode::INTERNAL_SERVER_ERROR, "no users").into_response(),
    };
    let workspaces = match state.workspaces.clone() {
        Some(w) => w,
        None => return (StatusCode::INTERNAL_SERVER_ERROR, "no workspaces").into_response(),
    };
    let sessions = match state.sessions.clone() {
        Some(s) => s,
        None => return (StatusCode::INTERNAL_SERVER_ERROR, "no sessions").into_response(),
    };

    let flow = match read_flow_cookie(&headers) {
        Some(f) => f,
        None => return err(StatusCode::BAD_REQUEST, "auth.oidc.state_mismatch", "missing flow"),
    };
    if flow.state != q.state {
        return err(StatusCode::BAD_REQUEST, "auth.oidc.state_mismatch", "state mismatch");
    }

    let id = match oidc.exchange_code(&q.code, &flow.pkce, &flow.nonce).await {
        Ok(i) => i,
        Err(e) => {
            tracing::warn!(error=?e, "oidc exchange");
            return err(StatusCode::BAD_REQUEST, "auth.oidc.exchange_failed", "");
        }
    };

    // Resolve existing user by (issuer, subject) first, then email.
    let user = match users.find_by_oidc(&oidc.issuer, &id.subject).await {
        Ok(Some(u)) => u,
        Ok(None) => match users.find_by_email(&id.email).await {
            Ok(Some(u)) => u,
            Ok(None) => match auto_provision(&state, &id, &users).await {
                Ok(Some(u)) => u,
                Ok(None) => return err(
                    StatusCode::FORBIDDEN,
                    "auth.oidc.not_provisioned",
                    "user not provisioned",
                ),
                Err(resp) => return resp,
            },
            Err(e) => {
                tracing::error!(error=?e, "oidc lookup");
                return err(StatusCode::INTERNAL_SERVER_ERROR, "internal", "");
            }
        },
        Err(e) => {
            tracing::error!(error=?e, "oidc lookup");
            return err(StatusCode::INTERNAL_SERVER_ERROR, "internal", "");
        }
    };

    // Workspace membership.
    let ws = match workspaces.get_singleton().await {
        Ok(Some(w)) => w,
        _ => return err(StatusCode::INTERNAL_SERVER_ERROR, "internal", ""),
    };
    if workspaces.get_member_role(ws.id, user.id).await.ok().flatten().is_none() {
        // Add as viewer (group mapping handled inside auto_provision; if user
        // already existed before OIDC was enabled, we add them as viewer here).
        if let Err(e) = workspaces
            .add_member(ws.id, user.id, knot_storage::WorkspaceRole::Viewer)
            .await
        {
            tracing::error!(error=?e, "oidc add_member");
            return err(StatusCode::INTERNAL_SERVER_ERROR, "internal", "");
        }
    }

    let token = SessionToken::generate();
    let exp = Utc::now() + Duration::days(30);
    if let Err(e) = sessions
        .create(token.as_bytes(), user.id, ws.id, exp, None, None)
        .await
    {
        tracing::error!(error=?e, "oidc create session");
        return err(StatusCode::INTERNAL_SERVER_ERROR, "internal", "");
    }

    let secure = state.base_url.starts_with("https://");
    let sid_cookie = format!(
        "{SID_COOKIE}={}; HttpOnly; SameSite=Lax; Path=/{}",
        token.encode(),
        if secure { "; Secure" } else { "" }
    );
    let flow_cleared = format!("{FLOW_COOKIE}=; HttpOnly; SameSite=Lax; Path=/; Max-Age=0");

    let mut resp = Redirect::to(&state.base_url).into_response();
    resp.headers_mut()
        .append(header::SET_COOKIE, HeaderValue::from_str(&sid_cookie).unwrap());
    resp.headers_mut()
        .append(header::SET_COOKIE, HeaderValue::from_str(&flow_cleared).unwrap());
    resp
}

async fn auto_provision(
    state: &AppState,
    id: &knot_auth::oidc::VerifiedIdentity,
    users: &std::sync::Arc<dyn knot_storage::UserStore>,
) -> Result<Option<knot_storage::User>, axum::response::Response> {
    let policy = std::env::var("KNOT_OIDC_AUTO_PROVISION").unwrap_or_else(|_| "off".into());
    // For Plan 3 we read directly from env. Future: thread Config through AppState.
    let allow = match policy.as_str() {
        "always" => true,
        "domain" => {
            let domains = std::env::var("KNOT_OIDC_ALLOWED_DOMAINS").unwrap_or_default();
            let user_domain = id.email.split('@').nth(1).unwrap_or("");
            domains
                .split(',')
                .map(str::trim)
                .filter(|s| !s.is_empty())
                .any(|d| d == user_domain)
        }
        "group" => {
            let mapping = std::env::var("KNOT_OIDC_ROLE_FROM_GROUPS").unwrap_or_default();
            let parsed: HashMap<String, String> =
                serde_json::from_str(&mapping).unwrap_or_default();
            id.groups.iter().any(|g| parsed.contains_key(g))
        }
        _ => false,
    };
    if !allow {
        return Ok(None);
    }

    let oidc = match state.oidc.as_ref() {
        Some(o) => o,
        None => return Err((StatusCode::INTERNAL_SERVER_ERROR, "no oidc").into_response()),
    };
    let created = users
        .create_oidc(&id.email, &id.display_name, &oidc.issuer, &id.subject)
        .await
        .map_err(|e| (StatusCode::INTERNAL_SERVER_ERROR, format!("{e}")).into_response())?;

    // If "group" policy resolved a role, attach as workspace member with that role.
    if policy == "group" {
        let mapping = std::env::var("KNOT_OIDC_ROLE_FROM_GROUPS").unwrap_or_default();
        let parsed: HashMap<String, String> =
            serde_json::from_str(&mapping).unwrap_or_default();
        let role = id
            .groups
            .iter()
            .find_map(|g| parsed.get(g))
            .and_then(|s| knot_storage::WorkspaceRole::parse(s))
            .unwrap_or(knot_storage::WorkspaceRole::Viewer);
        if let Some(ws_store) = state.workspaces.clone() {
            if let Ok(Some(ws)) = ws_store.get_singleton().await {
                let _ = ws_store.add_member(ws.id, created.id, role).await;
            }
        }
    }
    Ok(Some(created))
}

fn err(s: StatusCode, code: &str, msg: &str) -> axum::response::Response {
    (
        s,
        Json(json!({"error": {"code": code, "message": msg, "details": {}}})),
    )
        .into_response()
}

fn read_flow_cookie(headers: &HeaderMap) -> Option<FlowState> {
    let h = headers.get(header::COOKIE)?.to_str().ok()?;
    for raw in h.split(';') {
        if let Ok(c) = cookie::Cookie::parse(raw.trim()) {
            if c.name() == FLOW_COOKIE {
                let bytes = base64::engine::general_purpose::URL_SAFE_NO_PAD
                    .decode(c.value())
                    .ok()?;
                return serde_json::from_slice(&bytes).ok();
            }
        }
    }
    None
}
```

```rust
//! `knot-server admin create` — headless first-user bootstrap.

use std::io::{Read, Write};
use std::sync::Arc;

use clap::{Args, Subcommand};
use knot_auth::Hasher;
use knot_config::Config;
use knot_storage::{
    PgSessionStore, PgUserStore, PgWorkspaceStore, UserStore, WorkspaceRole, WorkspaceStore,
};

#[derive(Args)]
pub struct AdminArgs {
    #[command(subcommand)]
    pub cmd: AdminCmd,
}

#[derive(Subcommand)]
pub enum AdminCmd {
    /// Create the first user (and the singleton workspace if missing).
    /// Reads the password from stdin (so it stays out of shell history).
    Create {
        #[arg(long)]
        email: String,
        #[arg(long)]
        display_name: String,
        /// Workspace name to use when bootstrapping. Ignored if a workspace
        /// already exists.
        #[arg(long, default_value = "Workspace")]
        workspace_name: String,
        /// Workspace slug used at creation.
        #[arg(long, default_value = "default")]
        workspace_slug: String,
    },
}

pub async fn run(cfg: Config, args: AdminArgs) -> anyhow::Result<()> {
    match args.cmd {
        AdminCmd::Create {
            email,
            display_name,
            workspace_name,
            workspace_slug,
        } => create(cfg, &email, &display_name, &workspace_name, &workspace_slug).await,
    }
}

async fn create(
    cfg: Config,
    email: &str,
    display_name: &str,
    workspace_name: &str,
    workspace_slug: &str,
) -> anyhow::Result<()> {
    if cfg.database_url.is_empty() {
        anyhow::bail!("KNOT_DATABASE_URL must be set");
    }
    let pool = knot_storage::connect(&cfg.database_url, 4).await?;
    let users = Arc::new(PgUserStore::new(pool.clone()));
    let ws = Arc::new(PgWorkspaceStore::new(pool.clone()));
    let _sessions = PgSessionStore::new(pool);

    let mut buf = String::new();
    write!(std::io::stderr(), "password (read from stdin): ").ok();
    std::io::stderr().flush().ok();
    std::io::stdin().read_to_string(&mut buf)?;
    let password = buf.trim_end_matches(['\n', '\r']);
    if password.len() < 8 {
        anyhow::bail!("password must be at least 8 characters");
    }

    let count = users.count().await?;
    if count > 0 {
        anyhow::bail!("users already exist; admin create is first-run only");
    }

    let hasher = Hasher::new();
    let hash = hasher.hash(password).map_err(|e| anyhow::anyhow!(e.to_string()))?;

    let workspace = match ws.get_singleton().await? {
        Some(w) => w,
        None => ws.create(workspace_slug, workspace_name).await?,
    };
    let user = users.create_local(email, display_name, &hash).await?;
    ws.add_member(workspace.id, user.id, WorkspaceRole::Owner).await?;

    println!(
        "created user {} ({}) as owner of workspace {} ({})",
        user.id, email, workspace.id, workspace.slug,
    );
    Ok(())
}
```
